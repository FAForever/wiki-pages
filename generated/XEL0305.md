"Percival" is a UEF seabed amphibious unit included in *Forged Alliance Forever*.
It is classified as a tech 3 armored assault bot unit.
The build description for this unit is:

<blockquote>Extremely Heavy Amphibious Assault Robot. Designed to engage enemy Experimental units, heavy units, and defense structures.</blockquote>
