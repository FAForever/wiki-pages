"Deceiver" is a Cybran land unit included in *Forged Alliance Forever*.
It is classified as a tech 2 mobile stealth field system unit.
The build description for this unit is:

<blockquote>Mobile Stealth Field Generator. The field is active even inside of Transports, hiding all loaded units.</blockquote>
