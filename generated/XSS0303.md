"Iavish" is a Seraphim naval unit included in *Forged Alliance Forever*.
It is classified as a tech 3 aircraft carrier unit.
The build description for this unit is:

<blockquote>Aircraft Carrier. Can store, transport and repair aircraft. Armed with powerful AA auto-cannons and a tactical missile launcher. Equipped with a tactical missile defense system and a full sensor suite.</blockquote>
