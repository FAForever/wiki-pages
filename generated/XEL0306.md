"Spearhead" is a UEF land unit included in *Forged Alliance Forever*.
It is classified as a tech 3 mobile missile platform unit.
The build description for this unit is:

<blockquote>Heavy Mobile Missile Launcher, designed to overwhelm enemy shields and tactical missile defenses with large salvos.</blockquote>
