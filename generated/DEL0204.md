"Mongoose" is a UEF land unit included in *Forged Alliance Forever*.
It is classified as a tech 2 gatling bot unit.
The build description for this unit is:

<blockquote>Long range Support Robot. Armed with a Gatling Cannon and Fragmentation Grenades.</blockquote>
