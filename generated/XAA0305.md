"Restorer" is an Aeon aircraft unit included in *Forged Alliance Forever*.
It is classified as a tech 3 aa gunship unit.
The build description for this unit is:

<blockquote>Versatile Assault Gunship. Armed with a direct fire cannon and powerful anti-air rockets.</blockquote>
