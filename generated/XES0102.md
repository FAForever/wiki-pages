"Cooper" is a UEF naval unit included in *Forged Alliance Forever*.
It is classified as a tech 2 torpedo boat unit.
The build description for this unit is:

<blockquote>Lightly armoured, dedicated submersible killing boat, armed with a torpedo tube and torpedo defense. Vulnerable to direct fire.</blockquote>
