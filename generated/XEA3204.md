"C-D2 "Rover-2"" is a UEF aircraft unit included in *Forged Alliance Forever*.
It is classified as a tech 2 engineering drone unit.
This unit has no categories to define common builders, however the build description for it is:

<blockquote>Flying Drone, which can assist, repair, and reclaim any unit. Cannot be ordered to initiate the construction of structures. Vulnerable to any air fighters or anti-air units.</blockquote>
