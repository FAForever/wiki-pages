"Spook" is a Cybran aircraft unit included in *Forged Alliance Forever*.
It is classified as a tech 3 spy plane unit.
The build description for this unit is:

<blockquote>Extremely fast Stealth Spy Plane, with a large visual radius, good radar, and a small omni sensor.</blockquote>
