"Ithalua" is a Seraphim naval unit included in *Forged Alliance Forever*.
It is classified as a tech 2 cruiser unit.
The build description for this unit is:

<blockquote>Anti-Aircraft naval vessel. Armed with an anti-air flak-system, an anti-air Gatling gun and a tactical missile launcher. Equipped with a tactical missile defense system and a full sensor suite.</blockquote>
