This unnamed unit is a UEF aircraft unit included in *Forged Alliance Forever*.
It is classified as a defense satellite unit.
This unit has no categories to define common builders, however the build description for it is:

<blockquote>Satellite is untargetable by any weapon, but can attack enemy units and structures with very high precision. It can be destroyed only if its control center is destroyed.</blockquote>
