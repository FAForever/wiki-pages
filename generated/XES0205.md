"Bulwark" is a UEF naval unit included in *Forged Alliance Forever*.
It is classified as a tech 2 shield boat unit.
The build description for this unit is:

<blockquote>Naval Shield Generator, which provides protection to units beneath its large shield. Shields transfer residual damage to intersecting bubbles.</blockquote>
