"Cougar" is a UEF land unit included in *Forged Alliance Forever*.
It is classified as a tech 3 mobile rapid-fire aa cannon unit.
The build description for this unit is:

<blockquote> Mobile Anti-Air unit. Armed with a rapid-fire cannon, excellent against enemy gunships or air-fighters.</blockquote>
