"Stinger" is a UEF aircraft unit included in *Forged Alliance Forever*.
It is classified as a tech 2 gunship unit.
The build description for this unit is:

<blockquote>Light Gunship, equipped with a single transportation clamp. Can carry a single T1 or T2 unit.</blockquote>
