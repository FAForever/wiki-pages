"The Kennel" is a UEF structure unit included in *Forged Alliance Forever*.
It is classified as a tech 3 engineering station unit.
This unit has no categories to define common builders, however the build description for it is:

<blockquote>Houses two flying Drones, each of which automatically assists or repairs any unit with its operational radius. Can be ordered to fly around and reclaim, but cannot initiate construction.</blockquote>
