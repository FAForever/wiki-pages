"Serenity" is an Aeon land unit included in *Forged Alliance Forever*.
It is classified as a tech 3 mobile heavy artillery unit.
The build description for this unit is:

<blockquote>Slow-moving Heavy Artillery. Can fire while moving.</blockquote>
