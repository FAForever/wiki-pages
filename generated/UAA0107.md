"Chariot" is an Aeon aircraft unit included in *Forged Alliance Forever*.
It is classified as a tech 1 light air transport unit.
The build description for this unit is:

<blockquote>Air Transport, which can carry 6 T1 units, 3 T2 units, or a single T3 unit (Plus 2 T1). Cannot transport the ACU.</blockquote>
