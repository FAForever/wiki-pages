"Spirit" is an Aeon hover unit included in *Forged Alliance Forever*.
It is classified as a tech 1 land scout unit.
The build description for this unit is:

<blockquote>Fast, lightly armoured, amphibious reconnaissance vehicle. Armed with a machine gun and a basic sensor suite.</blockquote>
