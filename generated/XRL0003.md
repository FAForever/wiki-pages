This unnamed unit is a Cybran structure unit included in *Forged Alliance Forever*.
It is classified as a tech 3 crab egg (brick) unit.
This unit has no categories to define common builders, however the build description for it is:

<blockquote>Extremely heavy Amphibious Assault Robot, equipped with direct fire blasters, weak torpedoes and anti-torpedo flares. Designed to engage enemy heavy units and defense structures.</blockquote>
